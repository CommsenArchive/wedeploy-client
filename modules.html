Project Modules

This project has declared the following modules:

Name Description
WeDeploy Java client :: API Module provides Java API (set of interfaces) to define contracts for accessing WeDeploy's APIs (set of remote services)
WeDeploy Java client :: cloud Module provides implementation of "cloud" part of "WeDeploy Java client :: API".
WeDeploy Java client :: data Module provides implementation of "data" part of "WeDeploy Java client :: API".
WeDeploy Java client :: mapper :: gson Module provides Gson based implementation of data mapper used by "WeDeploy Java client :: data".
WeDeploy Java client :: transport :: okhttp Module provides OkHttp based implementation of data transport used by all "WeDeploy Java client :: API" implementations.
WeDeploy Java client :: tests Tests for "WeDeploy Java client :: API" implementations.
WeDeploy Java client :: BOM BOM (Bill of Material) to be used by provisioning tools!
